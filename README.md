# SmartHome

## Architecture involves 

- Modbus RS485 modules for Sensors and Relay actuators
- ESP8266 as a Wifi gateway and REST/Websocket server
- Pc / RaspBerry as an app Web server
	- The logic stay here
- Android as a Mobile app client
